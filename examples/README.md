# Examples

This directory contains sample transcript and subtitle outputs generated by the project.

- `25FA_COMP1023-L02_250903.srt`: sample SRT subtitle output
- `25FA_COMP1023-L02_250903.txt`: sample plain-text transcript output
- `comp2211_batch/`: sample batch ASR outputs generated from multiple COMP2211 lecture videos on `RTX 5090`
- `video16_packy_benchmark/`: Packy ATR concurrency benchmark logs and `c8k10` refined `video16` output

Large media inputs and generated videos should not be committed to the repository.
